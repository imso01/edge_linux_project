## 🕹️ 오목 게임 (Omok Game)

**리눅스 프레임버퍼 + 클라이언트-서버 구조 기반의 4목 게임**

---

### 📌 프로젝트 개요

C 언어와 리눅스 환경에서 구현한 4목 게임입니다.  
**프레임버퍼를 이용한 UI 구성**과 **TCP 기반의 클라이언트-서버 통신 구조**를 바탕으로,  
양쪽 플레이어가 실시간으로 게임을 진행할 수 있도록 설계되었습니다.
